# Results Archive

- `raw_logs/` contains the supplied console logs.
- `summaries/` contains extracted JSON run summaries plus two summaries transcribed from the supplied message.
- `reported_llm_hbm_metrics.csv` combines all HBM/LLM results.
- `baseline_all_models.csv` contains every supplied ML baseline result.
- `benchmark_without_other_vaccine_history.csv` and `benchmark_with_other_vaccine_history.csv` provide the compact comparison tables used in the README.

The metrics are preserved as reported. They should not all be treated as controlled head-to-head comparisons because several runs use different splits and sampling policies.
